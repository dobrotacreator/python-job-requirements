6. **_–ö–∞–∫ —Å–¥–µ–ª–∞—Ç—å GROUP BY —Å –ø–æ–º–æ—â—å—é Django ORM?_**

7. **_–í —á–µ–º —Ä–∞–∑–ª–∏—á–∏–µ –º–µ—Ç–æ–¥–∞_** **_filter_** **_–∏_** **_exclude?_**
